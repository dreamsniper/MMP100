HTML

There are many Tags that are used in Html and Css. Since i am starting to
relearn how to do it why not make a website to assist others.

  • All HTML tags have an opening and a closing tag (tags are often reffered to as elements)
  • All visible elements are included between the body tags.
  • Some tags have empty space to distinguish the eye for readers.
  • HTML pages are text documnets
  • Opening tags can carry attributes which tell us more about the contents of that element.
  • Attributes require a name and a value.

Most of the HTML tags are meant to represent structure for a website.
When it comes to actual design of the page that can be done utilizing CSS.

COMMON TAGS

  1. The html tag denotes to the browser that it is an HTML file that is being read
  2. The p tag tells the browser that it is a
  3. The title tag adds information on the top of the browser window or on the tabs.
  4. The b tag makes the words in between bold.
  5. The i tag italisizes all the words in between.
  6. The br tag is used to skip text to the next line.
And of course these are just the tip of the iceberg.

SEMANTIC MARKUPS

While HTML is used to format the structure of a page semantic information can be added to place emphasis on words
or to give meaning to the reader of the code such as definitions or quotations.

  • Qoutations marks are a type of Symantic markup using the q tag to denote which words should be in quotes.
  • Block quotes are also another type of semantic markup, the blockquote tag is used for this markup.
  • The abbr is used for abbreviated words and acronyms.
  • The dfn tag is used to indicate the defining instance of a term
  • The address tag is used to store general contact information for the author of the page.
